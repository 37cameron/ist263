What is HTML?

Hypertext Markup Language or "HTML" is the standardized markup language used to display content on the World Wide Web. In simpler terms, HTML tells the web browser how to display text, images and other aspects of a website. Every time you visit a website, including this one, the majority of the formatting was created with HTML.

HTML is written using tags and elements. These bits of code tell the web browser what content is, and how it should be displayed.

What's a Tag?

Tags are little bites of HTML indicated using the forwards and backwards carrot symbols "<" and ">".

These tags signal to the web browser how the content inside the tags should be displayed. Text written within two tags will take the effect of those tags.

This is a heading tag.

<h1>

These are subheading tags.

<h2>, <h3>

This is a paragraph tag.

<p>

What's an element?

When a beginning and end tag are placed around a grouping of content, that's an element.

This is an Element.

<p>This website is super informative and awesome!</p>

Notice how the last tag has a backwards slash to dicate that it's the end tag.
